# LIVORA

Protótipo premium de uma plataforma de live stream, vídeo chat, matches e presentes digitais para criadores.

## Ficheiros
- `index.html` — plataforma principal
- `entrada_livora.html` — página de entrada/login
- `README.md` — descrição do projeto

## Publicar no GitHub Pages
Depois de enviares os ficheiros para o repositório:
Settings → Pages → Deploy from a branch → main → /root → Save.
